# Puppeteer demo

## 无头浏览器应用

-   页面自动化
-   网页监控
-   网络爬虫

## Puppeteer 特点

-   对网页进行截图保存为图片或 pdf
-   抓取单页应用(SPA)执行并渲染(解决传统 HTTP 爬虫抓取单页应用难以处理异步请求的问题)
-   做表单的自动提交、UI 的自动化测试、模拟键盘输入等
-   用浏览器自带的一些调试工具和性能分析工具帮助我们分析问题
-   谷歌团队维护
